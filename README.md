# 🩺 Dr. Semmelweis Handwashing Analysis

A data analysis project based on the historical work of **Dr. Ignaz Semmelweis**, who discovered that handwashing with chlorinated lime dramatically reduced childbed fever deaths in hospitals.

This project uses Python for data cleaning, exploratory data analysis (EDA), and visualization to demonstrate the impact of handwashing on maternal mortality.

---

## 📌 Project Objective

The goal of this project is to:

- Analyze historical birth and death records.
- Compare mortality rates before and after handwashing was introduced.
- Visualize trends using Python.
- Understand how data-driven decisions can save lives.

---

## 📂 Dataset (yearly_deaths_by_clinic.csv)

The dataset contains historical records from the Vienna General Hospital, including:

- Number of births
- Number of deaths
- Death percentage
- Date
- Clinic information

## 📌 Key Insights

- Maternal mortality was significantly higher before handwashing.
- After introducing mandatory handwashing, death rates declined sharply.
- The analysis demonstrates the importance of hygiene in healthcare.
- Historical data supports evidence-based medical practices.
